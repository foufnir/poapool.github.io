8. Enregistrement de notre stake pool

Dans cette partie du guide, nous allons ENFIN enregistrer notre stake pool.

Création du fichier metadata.json#

Afin d'enregistrer notre stake pool, nous devons fournir au réseau un fichier metadata.json, dans lequel nous stockerons toutes les informations pratiques concernant la stake pool. Cela comprend, le nom, l'adresse du site Web, le ticker, ainsi que la description.

Pour ce faire, nous devrons stocker ce fichier quelque part. Nous allons donc le stocker sur GitHub.

8.01 Si vous n'avez pas encore de compte GitHub, créez-en un ici. Si vous en avez déjà un, veuillez vous y connecter.

8.02 Cliquez maintenant sur l'icône + à droite, et sélectionnez New gist. 8.03 Donnez-lui le nom suivant:

pool-metadata.json

8.04 Maintenant, copiez-collez ceci dans le contenu du fichier:

{
"name": "TestPool",
"description": "The pool that tests all the pools",
"ticker": "TEST",
"homepage": "https://teststakepool.com"
}

8.05 Vous pouvez adapter le contenu du fichier pool-metadata.json pour qu'il corresponde à vos besoins. Le notre pourrait ressembler à ça:

{
"name": "POA is awesome!",
"description": "Go delegate to POA",
"ticker": "POA12",
"homepage": "https://poaisawesome.com"
}

8.06 Une fois que vous avez terminé (assurez-vous de vérifier plusieurs fois si vous ayez tout saisi correctement), vous pouvez maintenant cliquer sur le bouton vert Create secret gist.

8.07 Cliquez ensuite sur Raw

8.08 Maintenant, copiez le lien URL lorsque vous êtes sur la page RAW

8.09 Allez sur git.io, et collez l'URL, puis cliquez sur Shorten

8.10 Téléchargeons maintenant le fichier! Allez dans votre terminal, sortez du dossier dans lequel vous êtes, entrez dans le dossier pool-keys et téléchargez le fichier.

cd
cd cardano-node/pool-keys
note

Assurez-vous de remplacer TheUrlOfYourRawPage par l'URL DE VOTRE PAGE RAW, PAS L'URL ABRÉGÉE de l'étape 8.09.

Dans notre cas, nous utiliserons la commande suivante:

wget https://gist.githubusercontent.com/foufnir/193e9ae05a6c8646afbe3681b1f05f3b/raw/61de0c325e1de9bc1e0c66da33ddb4e259c93c58/pool-metadata.json
wget TheUrlOfYourRawPage

8.11 Exécutez maintenant la commande suivante:

cardano-cli shelley stake-pool metadata-hash --pool-metadata-file pool-metadata.json

Sur l'image ci-dessus, vous pouvez voir le hash du fichier indiqué par la flèche. Nous utiliserons ceci pendant le processus d'inscription.

Générer le certificat de la stake pool#

Nous allons maintenant générer le certificat. Assurez-vous de tout vérifier trois fois!

note

  • Remplacez YourPledgeInLovelaces, par le montant en Lovelaces dans votre portefeuille (payment.addr), que vous êtes prêt à engager comme pledge.

  • Remplacez YourFixedCostInLovelaces par le montant en Lovelaces que vous êtes prêt à facturer en tant que frais fixes. Le minimum est de 340 ADA, soit 340 millions de Lovelaces.

  • Remplacez YourPoolMarginInPercentage, par le montant de marge que vous êtes prêt à facturer en pourcentage, comme 0,01 par exemple pour des frais de 1%.

  • Remplacez TheIPaddressOfYourRelayNode, par l'adresse IP de votre serveur sur lesquels sont vos nœuds relais.

  • Remplacez TheURLYouShortened par l'URL que vous avez raccourcie sur git.io.

  • Remplacez TheHashOfYourPoolMetadataFile, par le hash de votre fichier pool-metadata.json que vous avez trouvé à l'étape 8.11.

Dans notre cas, nous utiliserons la commande suivante:

cardano-cli shelley stake-pool registration-certificate --cold-verification-key-file cold.vkey --vrf-verification-key-file vrf.vkey --pool-pledge 1000000 --pool-cost 340000000 --pool-margin 0.01 --pool-reward-account-verification-key-file stake.vkey --pool-owner-stake-verification-key-file stake.vkey --mainnet --pool-relay-ipv4 0.0.0.0 --pool-relay-port 3000 --metadata-url https://git.io/JULPb --metadata-hash 216960437b78187633d8dee93c1540939eb821d029e988b82946efe47a150d57 --out-file pool-registration.cert
cardano-cli shelley stake-pool registration-certificate --cold-verification-key-file cold.vkey --vrf-verification-key-file vrf.vkey --pool-pledge YourPledgeInLovelaces --pool-cost YourFixedCostInLovelaces --pool-margin YourPoolMarginInPercentage --pool-reward-account-verification-key-file stake.vkey --pool-owner-stake-verification-key-file stake.vkey --mainnet --pool-relay-ipv4 TheIPaddressOfYourRelayNode --pool-relay-port 3000 --metadata-url TheURLYouShortened --metadata-hash TheHashOfYourPoolMetadataFile --out-file pool-registration.cert

Générer votre certificat de délégation#

Afin de rendre notre pledge officiel, nous devons générer un certificat:

8.12 Exécutez la commande suivante pour générer le certificat:

cardano-cli shelley stake-address delegation-certificate --stake-verification-key-file stake.vkey --cold-verification-key-file cold.vkey --out-file delegation.cert

Calculer les frais liés à une transaction, construire une transaction et la soumettre#

8.13 Construisons un test de transaction, afin que nous puissions calculer les frais:

note

Assurez-vous de remplacer YourOwnUTxO par votre adresse UTxO. Vous ne vous en souvenez pas ou ne le trouvez pas? Voici la commande:

cardano-cli shelley query utxo --address $(cat payment.addr) --mainnet

Remplacez TheNumberOfTransactionsMadeTotheUTxO par le nombre de transactions effectuées sur l'UTxO. Vous pouvez le trouver en exécutant la commande ci-dessus. Assurez-vous de conserver le hashtag entre les deux!

Dans notre cas, nous utiliserons la commande suivante:

cardano-cli shelley transaction build-raw --tx-in 439784ebcfbfc5db241c5e6817a1a24ab5038f1f051cab25647108360694e9f6#0 --tx-out $(cat payment.addr)+0 --ttl 0 --fee 0 --out-file tx.raw --certificate-file pool-registration.cert --certificate-file delegation.cert
cardano-cli shelley transaction build-raw --tx-in YourOwnUTxO#TheNumberOfTransactionsMadeTotheUTxO --tx-out $(cat payment.addr)+0 --ttl 0 --fee 0 --out-file tx.raw --certificate-file pool-registration.cert --certificate-file delegation.cert

8.14 Calculer les frais de transaction:

cardano-cli shelley transaction calculate-min-fee --tx-body-file tx.raw --tx-in-count 1 --tx-out-count 1 --mainnet --witness-count 1 --byron-witness-count 0 --protocol-params-file protocol.json

Comme vous pouvez le voir ci-dessus, les frais dans notre cas seront de 183101 Lovelaces. Ce sera probablement différent pour vous. Assurez-vous donc de vérifier les frais dans les commandes suivantes.

8.15 Calculons de combien nous devons rembourser l'UTxO:

note

Assurez-vous de remplacer TheAmountOfLovelacesInYourUTxO par le montant en Lovelaces dans votre UTxO.

Et remplacez TheFeeInLovelaces par les frais que vous avez calculés à l'étape 8.14.

expr TheAmountInLovelacesInYourUTxO - 500000000 - TheFeeInLovelaces

8.16 Calculez le TTL et éditez la transaction:

cardano-cli shelley query tip --mainnet
note

Assurez-vous de remplacer slotNo par la valeur de slotNo que vous avez trouvée après l'exécution de la commande précédente.

Pour calculer le TTL, exécutez la commande suivante:

expr slotNo + 1000
note
  • Assurez-vous de remplacer YourOwnUTxO par votre adresse UTxO. Vous ne vous en souvenez pas ou ne le trouvez pas? Voici la commande:
cardano-cli shelley query utxo --address $(cat payment.addr) --mainnet

  • Remplacez TheNumberOfTransactionsMadeTotheUTxO par le nombre de transactions effectuées sur l'UTxO. Vous pouvez le trouver en exécutant la commande ci-dessus. Assurez-vous de conserve le hashtag!

  • Remplacez TheAmountToPayBackToYourUTxOWeCalculatedInStep8Dot15 par le montant en Lovelaces que vous avez calculé à l'étape 8.15.

  • Remplacez TheTTL par le TTL que vous avez calculé avec la commande précédente.

  • Remplacez TheFee par les frais en Lovelaces que vous avez calculés à l'étape 8.14.

cardano-cli shelley transaction build-raw --tx-in YourOwnUTxO#TheNumberOfTransactionsMadeTotheUTxO --tx-out $(cat payment.addr)+TheAmountToPayBackToYourUTxOWeCalculatedInStep8Dot15 --ttl TheTTL --fee TheFee --out-file tx.raw --certificate-file pool-registration.cert --certificate-file delegation.cert

8.17 Signez la transaction

cardano-cli shelley transaction sign --tx-body-file tx.raw --signing-key-file payment.skey --signing-key-file stake.skey --signing-key-file cold.skey --mainnet --out-file tx.signed

8.18 Et soumettez la au réseau !

cardano-cli shelley transaction submit --tx-file tx.signed --mainnet

8.19 Vérifiez maintenant votre pool ID:

cardano-cli shelley stake-pool id --verification-key-file cold.vkey

8.20 Allez maintenant sur PoolTool, et recherchez votre pool ID dans la barre de recherche. Si vous pouvez trouver votre stake pool, excellent travail! Si vous ne pouvez pas, attendez quelques minutes, revenez, et si elle n'apparait toujours pas... Vous avez probablement fait une erreur quelque-part, reprenez les étapes de cette partie du guide.
