9. Sauvegarde de vos fichiers

Nous en sommes maintenant à l'une des parties les plus importantes de ce guide. Vous ne voulez vraiment pas passer à côté. Nous allons sauvegarder nos fichiers sensibles, pour nous assurer que notre stake pool restera sûre et accessible en toutes circonstances.

Sauvegarde de vos fichiers#

9.01 Tout d'abord, ouvrez un nouvel onglet dans le terminal, et cette fois, ne vous connectez pas à votre VPS.

9.02 Exécutez maintenant la commande suivante pour télécharger le dossier pool-keys depuis votre VPS:

note

  • Remplacez YourUserNameOnYourLOCALcomputer par le nom d'utilisateur que vous utilisez sur votre propre ordinateur.

  • Remplacez YourUserNameOnYourVPS par le nom d'utilisateur que vous utilisez sur le VPS de votre noeud producteur de blocs.

  • Remplacez TheIPaddressOfYourVPS par l'adresse IP du VPS de votre noeud producteur de blocs.

  • Important: vous devrez peut-être ajuster le chemin (le chemin est par exemple /home/max/Documents/) si vous avez une structure de dossier différente sur votre ordinateur, ou si vous utilisez une langue différente sur votre PC.

Dans notre cas, voici à quoi ressemble la commande:

rsync -de "ssh -i /home/max/.ssh/myssh -p 3100" foufnir@I'm-not-giving-you-the-IP-address:/home/foufnir/cardano-node/pool-keys /home/max/Documents/
rsync -de "ssh -i /home/YourUserNameOnYourLOCALcomputer/.ssh/myssh -p 3100" YourUserNameOnYourVPS@TheIPaddressOfYourVPS:/home/YourUserNameOnYourVPS/cardano-node/pool-keys /home/YourUserNameOnYourLOCALcomputer/Documents/

9.03 Si tout s'est bien passé, vous devriez maintenant voir un dossier pool-keys dans votre dossier Documents sur votre ordinateur en local.

9.04 La tâche a effectué maintenant est simple mais incroyablement importante.

Commencez par déplacer le dossier pool-keys vers au moins deux appareils de stockage externe. Cela peut être une clé USB, une carte SD, un disque dur externe, ... Cela peut être tout, sauf les solutions de stockage qui sont connectées à Internet.

Ne pas les stocker sur votre ordinateur. Ne pas les stocker dans le cloud. Ne pas les stocker pas sur votre téléphone.

9.05 Une fois que vous avez fait cela, assurez-vous de vérifier plusieurs fois que tout est comme il se doit, déplacez le dossier pool-keys dans la corbeille, et videz-la.

9.06 Connectez-vous maintenant à votre VPS

Déléguez-vous à POA?

Sinon, pensez à le faire. C'est la seule façon pour nous de générer des fonds, fonds que nous pouvons ensuite utiliser pour investir dans nos activités en Afrique afin d'y faire la promotion de Cardano et d'en améliorer l'adoption, mais aussi pour créer davantage de guides comme celui que vous utilisez actuellement pour créer votre stake pool.

Notre ticker est POA. Vous savez désormais quoi faire.

Make the difference. #StakeWithAfrica

9.07 Entrez dans le dossier pool-keys et exécutez la commande suivante:

cd cardano-node/pool-keys
note

Important: avant d'exécuter la commande suivante, veuillez vérifier à nouveau que vous avez correctement stocké le dossier pool-keys au moins sur deux appareils de stockage externe. Parce qu'après ça, il n'y a pas de retour en arrière possible.

rm cold* delegation* pool* payment* stake* tx*
